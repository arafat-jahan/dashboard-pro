# Flutter Interactive Dashboard

**An interactive Flutter dashboard app** featuring animated cards, dynamic charts, and a draggable layout for analytics visualization.

---

##  Project Overview

This Flutter app demonstrates modern UI/UX design and interactive data visualization:

- **Animated cards** with tap and swipe gestures for detailed metrics.
- **Dynamic charts** (Line, Bar, Pie) using [`fl_chart`](https://pub.dev/packages/fl_chart) with smooth animations.
- **Draggable dashboard layout** for real-time metric reordering.

---

## ðŸ”¹ Features

### 1. Animated Cards
- Tap to **expand/collapse** cards to reveal detailed info.
- Swipe left/right to navigate cards.
- Metrics with progress bars and trend indicators.
- Smooth animations using `AnimatedContainer`, `AnimatedSize`, and `RotationTransition`.

### 2. Interactive Charts
- **Line Chart**, **Bar Chart**, and **Pie Chart** integration.
- Animated chart updates with **Randomize Data** button.
- Toggle between Line and Bar charts.
- Pie chart supports **touch interaction** for highlighting sections.

### 3. Customizable Layout
- Long press and drag cards to **reorder** metrics.
- Visual feedback with highlighted borders and shadows during drag.
- Fully responsive layout for different screen sizes.
